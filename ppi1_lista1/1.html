A diferença entre a WEB e a internet

Internet é o nome que se dá à rede que conecta os computadores ao redor do mundo já a web nada mais é do que o caminho que permite a você usufruir do conteúdo transferido pela internet.

Breve histórico da internet e web

Internet - quando surgiu?

A internet surgiu em 1969, com a criação da ARPANET, uma rede militar do Departamento de Defesa dos EUA, conectando universidades para comunicação durante a Guerra Fria.

Web - quando surgiu?

A Web surgiu em 1989, quando Tim Berners-Lee inventou a World Wide Web (WWW) no CERN.

  1. 1989 - Tim Berners-Lee propõe a WWW no CERN
  2. 1990: Berners-Lee escreve o primeiro cliente e servidor web e as especificações de protocolos como HTTP e HTML.
  3. 6 de agosto de 1991: A Web é disponibilizada ao público, com a apresentação do primeiro website.
  4. 1993: O CERN libera o código-fonte da World Wide Web para o domínio público, o que acelera sua adoção.
