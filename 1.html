O que é HTML?

O HTML é uma linguagem de marcação de hipertexto. Que foi criado com intuito cientifico no contexto universitário onde tentaram melhorar a documentação de pesquisas que iam se perdendo. Portanto, foi criado para trocas de informação de documentos. Atualmente esta é a linguagem característica da WEB

Estrutura de um documento HTML
  • <!DOCTYPE> -> O doctype é a definição do documento html
  • <html> -> O elemento representa a raiz do documento html
  • <head> -> o head é onde é manipulado todas as coleções metadadas do documento, como o link e o title
  • <meta charset> -> representa o atributo que define a decodificação do documento html, como por exemplo: utf-8, ascii e etc
  • <title> -> O title serve para dar o título ao documento, então na web seria o título do site no navegador.
  • <body> -> é corpo HTML, então é aqui onde será feito a parte do site, usando os elementos, tags, divs e etc. Aqui é feito então a parte esturutural do site
