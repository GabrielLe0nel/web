Formulários

O HTML possui a tag form que possibilita definir formulários para submissão de dados informados pelo usuário por meio de campos de input, dentro do formulário. a tag <form> possui além dos atributos globais e também atributo action="url", utilizado para definir o destino dos dados submetidos Exemplo: 

        
            <form action="url">
                <input type="text" name="variavelA"
                ...
            <form>

Tabelas

A tag <table> é utilizada para definição de tabela no HTML, intermente em uma tabela podemos criar linhas "rows" utilzando a tag <tr> e celulas dentro das linhas utilizando a tag <td> Exemplo:

        <table>
            <tr> 
                <td> Celula 1x1 </td> celula 1x2 </td>
                <td> Celula 2x1 </td> celula 2x2 </td>
            <tr> 
        </table>
O código anterior define a tabela abaixo
Celula 1x1 Celula 1x2
Celula 2x1 Celula 2x2

Além disso podemos manipular o número de espaços utilizado em linhas e colunas por uma célula utilizando rowspan e colspan
