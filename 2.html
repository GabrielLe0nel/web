Alguns comandos GIT

  • repositório git
  • git init - cria um repo em um diretório
  • git add - adicionar modificações do workcopy ao stage
  • git commit -m "asdf" - cria um commit a partir do stage
  • git log - exibe os commits do nosso repositório local
  • git status - exibe o status do workcopy (WK)
  • git checkout -b novabranch hash - cria uma nova branch apontando para a branch ou hash informado
  • git reset --hard - restaura os arquivos trackeados ao estado do HEAD
  • git push remote branch
  • git config user.name "Nome"
  • git config user.email "Nome@email.com"
  • git clone url - faz o clone de um repositório remoto
