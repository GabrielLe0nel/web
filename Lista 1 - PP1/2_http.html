O que é o protocolo HTTP e como é usado na web?

HTTP é um protocolo que permite a troca de arquivos, como documentos HTML. É a base de qualquer troca de dados na Web e um protocolo cliente-servidor, o que significa que as requisições são iniciadas pelo destinatário, geralmente um navegador da Web.

Na web ela é usada para transferência de dados, entre o cliente e servidor, por exemplo para carregar imagens, arquivos e etc


Campos request e response no HTTP

Uma requisição HTTP (HTTP request) é uma mensagem enviada por um cliente (como um navegador) a um servidor para solicitar ou enviar informações

Os códigos de status de resposta HTTP indicam se uma solicitação HTTP específica foi concluída com êxito. As respostas são agrupadas em cinco classes.

Alguns códigos de status famosos
  • Level 200, 202 (aceito), 200 (OK)
  • Level 400, 404 (não encontrado), 400 (Bad request)
  • Level 500, 202 (aceito), 200 (OK)
