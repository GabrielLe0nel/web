Explicação regra CSS

Uma regra CSS serve para dizer ao navegador como um elemento HTML deve aparecer.
Ou seja, ela define o estilo (como cor, tamanho, espaçamento etc.) de algo que está no seu site.

Como é formada uma regra CSS?

Uma regra CSS tem 3 partes principais:

  1. Seletor → diz qual elemento você quer modificar.
  2. Propriedade → diz o que você quer mudar (cor, tamanho, fonte, etc.).
  3. Valor → diz como você quer que fique (vermelho, 20px, negrito...).

Exemplo


        p {
            color: blue;
        }

explicação:

  • p → é o seletor, indica que queremos estilizar todos os parágrafos <p> do HTML.
  • color → é a propriedade, ou seja, o que vamos mudar.
  • blue → é o valor, ou seja, a cor que queremos aplicar, que no caso é a propriedade color do CSS.
