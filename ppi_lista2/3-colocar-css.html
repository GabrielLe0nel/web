Como colocar css no html?

São 4 formas de colocar estilização CSS no html

  1. Externo - por meio da tag <link rel="stylesheet" href="style.css"> isso dentro do elemento <head>
  2. incorporado - por meio da tag <style> no cabeçalho do nosso documento HTML
  3. Inline - por meio do atributo style em qualquer tag HTML. Exemplo: <p style="color: red;">lorem </p>
  4. ** por meio da árvore DOM via JavaScript (JS)
