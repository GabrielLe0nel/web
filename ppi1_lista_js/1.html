O que é JavaScript?

  1. A motivação para a criação do JavaScript.
    • O JavaScript foi criado no contexto da popularização da web, onde as páginas eram utilizadas CSS e HTML, então eram estáticas e não tinha interatividade. Portanto, o JS surgiu para dar dinamicidade para a web. A web precisava de uma linguagem leve, executada no navegador, que permitisse ações básicas sem recarregar a página — validação de formulários, animações simples, interação com o usuário etc.

  2. A diferença entre execução no navegador e no servidor.
    • No navegador, o JavaScript é executado no cliente, dentro do browser, manipulando a interface, o DOM e interagindo com o usuário. No servidor, o JavaScript é executado no back-end (como no Node.js), processando lógica do sistema, acessando banco de dados e respondendo requisições antes de enviar dados ao cliente.

  3. O conceito de sandbox no navegador.
    • Sandbox no navegador é um ambiente isolado onde o JavaScript roda com permissões limitadas. Ele impede que o código da página acesse diretamente o sistema do usuário ou recursos sensíveis, garantindo segurança e evitando que scripts maliciosos causem danos.

  4. Objetos globais em Js, document, console e alert();
    1. document - Objeto - Representa a página carregada. Permite acessar e modificar elementos HTML.
    2. console - Objeto - Usado para registrar mensagens e informações no console do navegador.
    3. alert() - Função - Exibe uma caixa de diálogo com uma mensagem.
